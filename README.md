# Week-3-Task.sol
 GDG AASTU – Week 3 Solidity Practice Task
Description:
A Solidity contract demonstrating arrays, structs, storage/memory/calldata, and mappings.
You can store people, update names, read data, and check if a name exists.

Features

Add a Person to an array

Save a favorite number in a mapping (name → favoriteNumber)

Update only the name of a person using storage

Retrieve a person struct from the array (view, memory)

Check if a name exists in the mapping
